# DOTA LOGGER
Dota logger lets you log your matches and shows you some key stats.

## How to use
Some sample data is provided if the site does not find anything stored in local storage.
- log matches by selecting the hero you played, the game length in MM:SS or HH:MM:SS, which side you played for and the result of the match.
- View, sort and edit the match data in the Matches list.
- The dashboard key stats update automatically with your match data.
